# 莲藕摆渡一体化心理赋能平台

一款聚焦刑满释放人员心理健康与全周期社会融入的一体化专业心理服务支撑系统。

## 一键部署到Vercel

点击下方按钮，即可将项目一键部署到Vercel云平台：

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/your-username/lotus-bridge-platform&project-name=lotus-bridge-platform&repository-name=lotus-bridge-platform)

## 部署步骤

1. 点击上方的「Deploy with Vercel」按钮
2. 登录您的Vercel账号（使用GitHub账号登录）
3. 配置项目信息：
   - 项目名称：`lotus-bridge-platform`
   - 部署区域：选择离您最近的区域
   - 环境变量：无需配置
4. 点击「Deploy」按钮开始部署
5. 部署完成后，您将获得一个公开访问URL

## 项目结构

```
.
├── dist/                 # 生产构建目录
│   ├── home1.html        # 官方网站主页
│   ├── index.html        # 平台管理后台
│   ├── dashboard.html    # 仪表盘页面
│   ├── user_management.html # 用户管理页面
│   ├── styles.css        # 样式文件
│   └── logo.webp         # 网站Logo
├── components/           # 可复用组件
├── vercel.json           # Vercel部署配置
└── README.md            # 项目说明文档
```

## 核心功能

- 首次测评与任务系统
- 课程体系
- 兴趣小组
- 藕粉积分机制
- 职业广场
- 心理帮扶

## 技术栈

- HTML5 + CSS3 + JavaScript
- Tailwind CSS v3.x
- Font Awesome
- Chart.js
- 响应式设计
- SEO优化
- 图片懒加载

## 开发流程

1. 克隆项目到本地
2. 在浏览器中打开 `dist/home1.html` 查看效果
3. 修改代码后刷新浏览器查看变更
4. 推送修改到GitHub
5. Vercel将自动部署更新
